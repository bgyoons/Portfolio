# **프로젝트 상세**
## 수정광산

<img src="./image/crystalmine.png"  width="300">

### 상세 내용

- 성신여대 학생들만이 이용할 수 있는 **성신여대 자체 커뮤니티** 앱 입니다.
- 성신여대를 생활하며 수강 신청, 주변 상권 등 필요한 정보를 한 곳에서 확인할 수 있도록 성신 위키를 제작하였습니다.
- 게시판 생성과 자유로운 글 작성, 스크랩을 통해 성신여대 동문들끼리의 소통을 할 수 있습니다.

### 사용 기술

- React-Native
- TypeScript

### 담당 기능
- UI 커스텀 컴포넌트
- 스플래시 화면
- 회원가입/회원탈퇴
- 게시판/게시글 검색, 검색 결과 조회
- 성신 위키 탭 게시판
- 문의하기/게시글 작성

### 깨달은 점
- React Native는 JavaScript로 되어있는 모바일 앱 개발 언어이다.  
- JavaScript와 React를 알면 Android와 iOS를 동시에 개발할 수 있는 장점이 있다. 
- 컴포넌트에 무한스크롤이나 새로고침을 위한 props를 제공해주어 편리하게 개발할 수 있다.

## CampBu

<img src="./image/campbu.png"  width="600">
<img src="./image/campbu1.png"  width="600">

### Repository & Wiki

- [Repository](https://github.com/codestates/Campbu)
- [위키](https://github.com/codestates/Campbu/wiki)

### 상세 내용

- 사용자 위치 주변에 있는 대여 가능한 캠핑 물품을 확인할 수 있습니다.
- 마음에 드는 캠핑 물품을 찜하기 버튼을 통해 목록에 저장할 수 있습니다.
- 대여 가능한 캠핑 물품을 예약할 수 있습니다.
- 대여 예약시 대여자와 실시간 채팅을 통해 상세한 대여 조건 및 대여 방법을 조율할 수 있습니다.
- 대여 후 캠핑물품 반납시 빌려준 사람, 빌린 사람에 대한 리뷰항목으로 작성할 수 있습니다.

### 사용 기술

- React 
- TypeScript
- Recoil
- Emotion.js

### 담당 기능

- 상품 검색 기능 구현
- 상품 목록 페이지 카테고리 기능, 상품 찜하기 구현
- 마이페이지, 메시지함 내 예약 및 대여 관리 기능 구현
- 예약 및 대여 상태에 따라 다른 모달창 구현
- 와이어프레임, 프로토타입 제작

### 깨달은 점

- TypeScript로 타입을 지정함으로써 자동완성이나 타입확인을 통해 빈 값이나 잘못된 타입, 오타 등 사전에 실수를 방지할 수 있었다.
- Recoil은 React와 호환이 뛰어나고 React Hook과 상당히 유사하다.
- Props drilling을 방지하고자 Recoil을 채택하였다.
- Emotion jsx에서 제공해주는 css 속성을 활용하면 이를 클래스로 변환해준다.
- css props를 결합하여 복잡한 스타일링을 적용하거나 css 변수를 조립하여 컴포넌트 스타일링을 적용할 수 있다.

- Styled-Component
